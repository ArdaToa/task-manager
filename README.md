# Görev Yöneticisi API

Görevleri yönetmek için Spring Boot ile oluşturulmuş RESTful bir API.

## Açıklama

Görev Yöneticisi API'si görev oluşturmak, okumak, güncellemek ve silmek (CRUD işlemleri) için uç noktalar sağlar. Görev takibi yapan uygulamalar için temel bir arka uç (backend) hizmeti olarak tasarlanmıştır.

## Kullanılan Teknolojiler

* Java
* Spring Boot
* Maven

## Başlarken

### Gereksinimler

* Java Development Kit (JDK) 17 veya daha yenisi
* Maven 3.6 veya daha yenisi

### Kurulum ve Çalıştırma

1. Projeyi bilgisayarınıza indirin (clone)
2. Proje dizinine gidin:
   `cd "Task Manager API"`
3. Maven kullanarak uygulamayı çalıştırın:
   `mvn spring-boot:run`

API `http://localhost:8080` adresinde çalışmaya başlayacaktır.

## API Uç Noktaları

(Belirli API uç noktaları geliştirildikçe buraya eklenebilir, örneğin: GET /api/tasks, POST /api/tasks vb.)
